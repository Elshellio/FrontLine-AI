Optional upgrade

Optional calendar and booking lookup

TableBoss does not need calendar lookup to recover missed calls and table requests. A calendar module can be added later for live availability lookup, booking views, pending confirmations and table planning.

This is an optional upgrade, not part of the initial live call-recovery dashboard. It can be added when the venue wants calendar-driven booking control.
